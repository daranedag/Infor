ForestSimulator Bwinpro Version 7.5 Region Norwestdeutschland

Diese Version ist für  MS Windows Betriebssysteme  und kann ohne Installation und ohne Administratorrechte eingesetzt werden.

Die JRE und und Java3D sind im Verzeichnis \jre6 vorhanden und müssen nicht installiert werden. 

Die Version kann auch von auf einem USB-Stick oder einer mobilen Platte eingesetzt werden. 

Einfach in ein beliebiges Verzeichnis entpacken.

Im Unterordner \JavaApps\ die Datei ForestSimulator.cmd anklicken. 


Für Linux müssen die JRE und Java3D installiert werden (siehe Handbuch).

Sie können den Simulator dann mit der Datei ForestSimulator.sh starten


Auf einem MAC sollte das Programm im 2D-Modus ohne Installation laufen
